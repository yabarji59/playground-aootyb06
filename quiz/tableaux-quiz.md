# Quiz : Les Tableaux en JavaScript

## Question 1
Quelle est la bonne façon de créer un tableau vide en JavaScript ?

- [ ] `let arr = [];`
- [ ] `let arr = new Array();`
- [x] Les deux réponses précédentes sont correctes
- [ ] `let arr = Array();`
